<task type="auto">
  <name>Task 1: Create Ingress resource with WebSocket annotations and path routing</name>
  <files>k8s/base/ingress/ingress.yaml</files>
  <action>
Create directory `k8s/base/ingress/`.

Create `k8s/base/ingress/ingress.yaml`:
- apiVersion: networking.k8s.io/v1
- kind: Ingress
- metadata:
  - name: multica-ingress
  - namespace: multica
  - labels:
    - app.kubernetes.io/name: multica-ingress
    - app.kubernetes.io/part-of: multica
  - annotations (ALL of these are CRITICAL for WebSocket support):
    - nginx.ingress.kubernetes.io/proxy-read-timeout: "3600"
    - nginx.ingress.kubernetes.io/proxy-send-timeout: "3600"
    - nginx.ingress.kubernetes.io/proxy-http-version: "1.1"
    - nginx.ingress.kubernetes.io/affinity: "cookie"
    - nginx.ingress.kubernetes.io/affinity-mode: "persistent"
    - nginx.ingress.kubernetes.io/session-cookie-name: "multica-sticky"
    - nginx.ingress.kubernetes.io/session-cookie-max-age: "3600"
- spec:
  - ingressClassName: nginx
  - rules (single rule, no host — set by overlay):
    - http:
      - paths:
        - path: /api
          pathType: Prefix
          backend:
            service:
              name: backend
              port:
                number: 8080
        - path: /
          pathType: Prefix
          backend:
            service:
              name: frontend
              port:
                number: 3000

IMPORTANT ordering: The /api path MUST come before / in the paths list. Nginx evaluates
paths by specificity but explicit ordering ensures correct behavior across ingress
controller versions.

Do NOT add a host field — this is the base manifest. Overlays will add the host.
Do NOT add TLS section — that is deferred to v2 (cert-manager).

Follow the same YAML formatting style as existing manifests (2-space indent, no trailing whitespace).
  </action>
  <verify>
    <automated>kustomize build k8s/base/ingress/ 2>&1 | head -5</automated>
  </verify>
  <done>
    - ingress.yaml exists with networking.k8s.io/v1 API version
    - All 5 WebSocket/affinity annotations are present
    - Two path rules: /api -> backend:8080, / -> frontend:3000
    - ingressClassName is nginx
    - No hardcoded host
  </done>
